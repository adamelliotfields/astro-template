All links are processed by the [`rehype-external-links`](https://github.com/rehypejs/rehype-external-links) plugin.

## Internal

Internal links point to pages generated at build time like <a href={import.meta.env.DEV ? 'http://localhost:4321' : `${import.meta.env.SITE}${import.meta.env.BASE_URL ?? ''}`}>home</a>, [blog](../../blog), [about](../../about), and [resume](../../resume).

Links are considered internal if they are relative or start with the `origin` from `src/config.json`.

## External

External links start with a `http`, `https`, `mailto`, or `tel` protocol. These links open a new browser tab via `target="_blank"` with `rel="noopener noreferrer"`.

An SVG external link icon is appended with `aria-hidden="true"` and `aria-label="opens in a new tab"`, so assistive technology users are made aware before clicking.
