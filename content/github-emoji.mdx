GitHub emoji or _Gemoji_ shortcodes are converted into Unicode emoji by [`remark-gemoji`](https://github.com/remarkjs/remark-gemoji). You can write familiar names like `:rocket:` or `:sparkles:` in Markdown and let the content pipeline render the final symbol.

## Examples

- Shipping a feature: :rocket:
- Polishing the UI: :sparkles:
- Fixing a bug: :bug:
- Writing docs: :memo:
- Celebrating a release: :tada:

See [emoji-cheat-sheet](https://github.com/ikatyang/emoji-cheat-sheet) for a reference generated by the [emojis](https://api.github.com/emojis) API endpoint.

## VS Code

The [MDX](https://marketplace.visualstudio.com/items?itemName=unifiedjs.vscode-mdx) extension will syntax highlight supported shortcodes via the [`gemoji`](https://github.com/wooorm/gemoji) package.
